# 🗂️ Meu Portfólio Dev

> Site pessoal que apresenta meus projetos e habilidades como desenvolvedor front-end em formação.

## 📋 Descrição

Site portfólio desenvolvido para reunir em um só lugar minha trajetória, habilidades técnicas, projetos e formas de contato — servindo como cartão de visita digital para recrutadores e empresas.

## ⚙️ O que o site faz

- Apresenta meus projetos com links de demo e repositório
- Exibe minhas habilidades técnicas com nível de proficiência
- Mostra minha trajetória profissional e acadêmica
- Permite contato direto via formulário, LinkedIn e WhatsApp
